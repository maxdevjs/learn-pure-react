## How it works

Set up to work on [codesandbox.io](https://codesandbox.io).
Offline should work too. From `package.json` `scripts` section:

```
"scripts": {
    "start": "react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test --env=jsdom",
```
